Bootstrap 4

Bootstrap 4是Bootstrap的最新版本，它是用于开发响应式，移动优先网站的最流行的HTML，CSS和JavaScript框架。 Bootstrap 4是完全免费的下载和使用！
Bootstrap 3 vs. Bootstrap 4：
Bootstrap 4是Bootstrap的最新版本;新组件，更快的样式表和更多的响应。 Bootstrap 4支持所有主流浏览器和平台的最新稳定版本。但是，Internet Explorer 9和不支持。
如果您需要IE8-9的支持，请使用Bootstrap 3，它是Bootstrap最稳定的版本，团队仍然支持关键错误修正和文档更改。但是，不会添加任何新功能。 注：Bootstrap 4目前处于测试版（意味着它已经过全面测试，接近“最终”版本）。
什么是Bootstrap？
Bootstrap是一个免费的前端框架，可以更快，更轻松地进行Web开发 Bootstrap包括基于HTML和CSS的排版，表单，按钮，表格，导航，模式，图像轮播等设计模板，以及可选的JavaScript插件 Bootstrap还使您能够轻松创建响应式设计。
tips–什么是响应式网页设计？
响应式网页设计是关于创建网站，自动调整自己在所有设备上看起来不错，从小型电话到大型桌面。
为什么用Bootstrap 4?
Bootstrap 4的优点：易于使用：任何只具有HTML和CSS基本知识的人都可以使用Bootstrap 4
响应功能：Bootstrap的响应式CSS可以调整到手机，平板电脑和桌面 移动优先方法：在Bootstrap 4中，移动优先方式是核心框架的一部分
浏览器兼容性：Bootstrap 4兼容所有现代浏览器（Chrome，Firefox，Internet Explorer 10+，Safari和Opera）
